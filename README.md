Device configuration for OnePlus 7 Pro

| CPU                     | Octa-core (1x2.84 GHz Kryo 485 & 3x2.42 GHz Kryo 485 & 4x1.80 GHz Kryo 485)                                                    |
| Chipset                 | Qualcomm SDM855 Snapdragon 855                                                                                                 |

| Memory                  | 6/8/12 GB RAM                                                                                                                  |
| Shipped Android Version | 9                                                                                                                              |
